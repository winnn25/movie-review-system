## Technical Features
- Database Management: Powered by a structured relational database (cinema_db) to ensure data consistency across ratings and reviews.
- Interactive UI: A dark-themed, modern interface (as seen in the screenshots) designed for high readability and easy navigation.
- Security: Includes role-based logic to protect administrative functions and sensitive user data.

## Project Structure
- Admin Dashboard: Centralized hub for managing movies, user profiles, and system-wide feedback.
- User Portal: Personalized dashboard for exploring content and managing individual ratings/reviews.
